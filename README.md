# E-Commerce Conversion Rate Analysis

## Objective
Analyze customer behavior and measure the percentage of visitors who complete a purchase.

## Tools Used
- Tableau
- Excel Dataset

## KPIs
- Total Visitors
- Total Purchases
- Conversion Rate

## Visualizations
- Conversion Trend
- Traffic Source Analysis
- Visitor Type Analysis
- Device Analysis

## Outcome
Built an interactive dashboard to identify factors affecting e-commerce conversions.
